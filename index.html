О нас

Это Telegram Web App для оперативного освещения КХЛ и ВХЛ.

Показываем матчи, которые идут прямо сейчас, и расписание на завтра.

Время отображается по Москве (MSK).

Источники данных: публичный API SofaScore + статьи из репозитория (articles.txt).
